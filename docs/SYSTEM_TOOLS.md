# System Tools

Tools for disk management, recovery, and partitioning.

## Download Links (run scripts/downloadIsos.sh)

- **GParted** - Partition editor (Live ISO)
- **SystemRescue** - System rescue toolkit
- **Clonezilla** - Disk imaging/cloning
- **DBAN** - Disk wipe/nuke (for secure erase)
- **Parted Magic** - Partition management (paid, check for free alternatives)
- **Knoppix** - Live CD for recovery

## Usage

These are typically .iso files that can be booted via Ventoy.

For DBAN: Use the .iso to securely erase hard drives before disposal.
For GParted: Boot the ISO to resize, create, or manage partitions.
